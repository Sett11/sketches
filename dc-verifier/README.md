# DCV (Data Chains Verifier)

Инструмент для проверки целостности цепочек данных между слоями приложения (Frontend → Backend → Database) **БЕЗ ЗАПУСКА приложения**.

## Концепция

DCV анализирует поток данных через граф вызовов:
1. Находит точку входа (main.py, app.py)
2. Строит граф вызовов: отслеживает импорты → функции → классы → методы
3. Идет по графу: main → route → handler → crud → model
4. Отслеживает поток данных через граф
5. Проверяет контракты на каждом стыке
6. Формирует отчет о несоответствиях

## Установка

```bash
cargo build --release
```

## Использование

### Инициализация конфига

```bash
dc-verifier init
```

### Проверка цепочек

```bash
dc-verifier check
```

## Структура проекта

- `crates/dc-core/` - Ядро: построение графов, анализ потока данных
- `crates/dc-adapter-fastapi/` - Адаптер для FastAPI
- `crates/dc-typescript/` - Адаптер для TypeScript
- `crates/dc-cli/` - CLI инструмент
